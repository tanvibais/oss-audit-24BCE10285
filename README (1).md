## 📁 What’s Inside This Repository?

### 🖥️ 1. System Identity Report
File: System-Identity-Report.sh

This script shows basic system information like user, uptime, kernel version, date, and Linux distribution.

---

### 📦 2. FOSS Package Inspector
File: Foss-Package-Inspector.sh

Checks whether Python3 is installed and shows its version.

---

### 📊 3. Log File Analyzer
File: Log-file-analyzer.sh

Analyzes a log file, counts keyword occurrences (default: "error"), and shows last 5 matching lines.

---

### ✍️ 4. Open Source Manifesto Generator
File: open-source-manifesto-generator.sh

Takes user input and generates a personal open-source manifesto file.

---

### 📄 5. Sample Manifesto
File: manifesto_tanvi.txt

A sample output file generated from the manifesto script.

## ⚙️ Requirements

- Linux OS / WSL
- Bash shell
- Commands: grep, tail, uname, whoami, date, uptime, which
- Optional: lsb_release
- Python3

## ▶️ How to Run

### Step 1: Clone repository
git clone <your-repo-link>
cd <repo-folder>

### Step 2: Give permission
chmod +x *.sh

### Step 3: Run scripts

System Report:
./System-Identity-Report.sh

Package Inspector:
./Foss-Package-Inspector.sh

Log Analyzer:
./Log-file-analyzer.sh <logfile> [keyword]

Manifesto Generator:
./open-source-manifesto-generator.sh
